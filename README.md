# Golang MongoDB Implementation

This project has been prepared for the purpose of example of MongoDB integration with Golang v1.18

## Libraries

1. globalsign/mgo
2. spf13/viper
3. gin-gonic/gin

## Installing

```
make run (if you has make) or go run main.go
```
